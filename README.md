# 📖 تطبيق مصحف - Flutter

تطبيق إسلامي متكامل مبني بـ Flutter، مستوحى من تصميم HTML الأصلي مع تحسينات شاملة.

---

## ✨ المزايا

| الصفحة | المزايا |
|--------|---------|
| 🕌 **الصلاة** | مواقيت الصلاة عبر API • عداد تنازلي للصلاة القادمة • تحديد الصلاة الحالية |
| 📖 **القرآن** | قائمة كل السور الـ 114 • بحث بالعربية والإنجليزية • عرض الآيات مع التحميل |
| 🤲 **الأذكار** | أذكار الصباح والمساء والنوم • عداد لكل ذكر • شريط تقدم بصري |
| 📿 **التسبيح** | 8 أنواع من الأذكار • حلقة تقدم مرئية • تأثيرات احتفالية عند الإتمام |
| ☪️ **الأسماء** | 99 اسماً من أسماء الله • شبكة تفاعلية • نافذة منبثقة بالمعنى والفائدة |
| 💛 **ادعمنا** | صفحة دعم مع حديث شريف |

---

## ⚡ التأثيرات التفاعلية

- **انيميشن انتقال** بين الصفحات (Fade + Slide)
- **Nav bounce** عند تحديد تبويب جديد
- **Scale pulse** عند الضغط على أي عنصر
- **حلقة دائرية** متحركة في صفحة التسبيح
- **Staggered fade** لقائمة الآيات والأسماء
- **Bottom sheet** بانيميشن للأسماء الحسنى
- **تحريك بالحجم** عند تحديث عداد الأذكار
- **Elastic bounce** عند إكمال التسبيح
- **الهجري** محسوب محلياً بدون API

## 🚀 خطوات التشغيل

### 1. تثبيت Flutter
```bash
flutter --version  # تأكد من التثبيت
```

### 2. إنشاء مشروع جديد
```bash
flutter create mushaf_app
cd mushaf_app
```

### 3. استبدال الملفات
- انسخ `lib/main.dart` إلى مجلد `lib/` في مشروعك
- انسخ `pubspec.yaml` أو أضف dependency واحدة:

```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.1
```

### 4. تثبيت packages
```bash
flutter pub get
```

### 5. تشغيل التطبيق
```bash
flutter run
```

## 🎨 التصميم

- **ألوان**: خلفية داكنة + ذهبي (#C9A84C) - نفس التطبيق الأصلي
- **خطوط**: Amiri للعربية + Cairo للنص
- **RTL**: دعم كامل للغة العربية
- **الكود الهجري**: محسوب محلياً (بدون API)
