# مشروع مصحف (Flutter) 🕌

تطبيق إسلامي شامل تم تحويله من HTML إلى Flutter لدعم Android و iOS.

## الميزات الرئيسية
- **القرآن الكريم:** تلاوة صوتية مع نظام تكرار متقدم للحفظ.
- **مواقيت الصلاة:** جلب تلقائي عبر GPS مع عد تنازلي حي.
- **الأذكار والتسابيح:** عدادات تفاعلية مع حفظ التقدم.
- **أسماء الله الحسنى:** عرض الـ 99 اسماً مع المعاني والبحث.

## هيكل الملفات
- `lib/services/`: يحتوي على خدمات الـ API والصوت والموقع.
- `lib/screens/`: الشاشات الرئيسية للتطبيق.
- `lib/utils/`: بيانات الأذكار والسور والأسماء.

## الـ APIs المستخدمة
- [Aladhan API](https://aladhan.com/prayer-times-api) لمواقيت الصلاة.
- [Alquran Cloud](https://alquran.cloud/api) للآيات والتلاوات.

## التشغيل
1. قم بتنفيذ `flutter pub get`.
2. تأكد من إضافة الخطوط في مجلد `assets/fonts/`.
3. شغل التطبيق باستخدام `flutter run`.
